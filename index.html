听见主、看见主、摸到主

发表于 2018-10-21   |   分类于 圣经

论到从起初原有的生命之道，就是我们所听见、所看见、亲眼看过、亲手摸过的。（ 约翰一书 1:1 和合本)

这节经文至少描述了一个人与主的三个阶段。一，听见主二，看见主三，摸过主。

我们基督徒的生命，大致也有这三个阶段。

  1. 开始我们听到有人传福音给我们，然后可能百般无奈之下，或者碍于别人的面子，信了。

  2. 然后可能聚会都挺无奈的，结果突然有一天看到了主的作为，开始真正的相信。

  3. 因为相信，经历主越来越多。慢慢的，可能就信仰陷于疲软。然后神引领你到旷野深处，水深之处，去单独与神会晤和谈话。

被神呼召拯救之人，进入到深处之时，都会有这个体悟，那时候的环境犹如旷野，只有上帝在荆棘之中。犹如摩西，犹如亚伯拉罕，犹如以利亚，犹如保罗…这些圣徒在旷野摸到主了。开始被训练，被陶造，预备成为合用的器皿。每个基督里的人都有如此过程，也是让我们明白自己在哪里？要来得什么？

对“顺服”的认识

发表于 2018-10-19   |   分类于 圣经 ， 成圣

顺服的经文

你们作仆人的，凡事要存敬畏的心顺服主人；不但顺服那善良温和的，就是那乖僻的也要顺服。倘若人为叫良心对得住　神，就忍受冤屈的苦楚，这是可喜爱的。你们若因犯罪受责打，能忍耐，有什么可夸的呢？但你们若因行善受苦，能忍耐，这在　神看是可喜爱的。你们蒙召原是为此；因基督也为你们受过苦，给你们留下榜样，叫你们跟随他的脚踪行。他并没有犯罪，口里也没有诡诈。他被骂不还口；受害不说威吓的话，只将自己交托那按公义审判人的主。他被挂在木头上，亲身担当了我们的罪，使我们既然在罪上死，就得以在义上活。因他受的鞭伤，你们便得了医治。你们从前好像迷路的羊，如今却归到你们灵魂的牧人监督了。 (彼得前书 2:18-25 和合本)

你们作妻子的要顺服自己的丈夫；这样，若有不信从道理的丈夫，他们虽然不听道，也可以因妻子的品行被感化过来；这正是因看见你们有贞洁的品行和敬畏的心。 (彼得前书 3:1-2 和合本)

顺服的内容

今天的经文讲的是『顺服』。『顺服』是基督徒非常重要的一个属灵品格。我们从总原则上讲，我们顺服任何的次序和权柄，都是因为次序和权柄是神所赐的，因为国度，权柄，荣耀全是神的，直到永远。

为此，我们操作『顺服』的时候，会有很多具体的内容。

  • 一，从年龄的角度来说，年幼的要顺服年长的。这个包含有血缘关系和无血缘关系的人。家中的父母，长辈是要顺服的，这个特别在教导孩子的时候，是需要特别重点关注的。对于成人也是如此，对父母长辈，在无关乎圣经原则的事情上，要学习顺服。在社会的层面，对年长的人要尊敬，意见要考虑，言语要柔和，这些都是基督徒需要实际操作的，且是为了神的缘故。
  • 二，妻子顺服丈夫。妻子要学习顺服丈夫，不能说丈夫爱我我就顺服，丈夫对我不好我就不顺服，这是错误的教导。妻子顺服丈夫，是因为丈夫是妻子的头，为此顺服丈夫是因为主的缘故。在具体操作上，妻子需要在不违背圣经的基础上，原则无条件顺服丈夫，为此必然蒙属灵的祝福，也有更多建立家室的机会。
  • 三，仆人顺服主人。这点落实到现在的时候，就是类似员工顺服老板，职位低的顺服职位高的。这块内容也是职场传福音的很重要的内容，需要有见证。也特别在现在经常跳槽和造反的年代，这点尤为宝贵。
  • 四，对执政者的顺服。在不违背圣经原则的基础上，基督徒需要顺服执政者所设立的法律，对执政者要尊敬。
  • 五，对教会的顺服。基督徒不能做野羊，不服教会的约束，自己想干嘛就干嘛，没有身体的意识，觉得自己和主关系很好。这是很可怕的，群羊需要跟随牧人，记得是『群羊』。

基督徒如何看“做计划”

发表于 2018-10-18   |   分类于 圣经

嗐！你们有话说：“今天明天我们要往某城里去，在那里住一年，做买卖得利。”其实明天如何，你们还不知道。你们的生命是什么呢？你们原来是一片云雾，出现少时就不见了。你们只当说：“主若愿意，我们就可以活着，也可以做这事，或做那事。”现今你们竟以张狂夸口；凡这样夸口都是恶的。人若知道行善，却不去行，这就是他的罪了。 (雅各书 4:13-17 和合本)

计划

基督徒对计划的态度和世人不同。世人有句话说『计划赶不上变化』，这句话是有主的普遍恩典在的。因为『变化』是在上帝手中。我们基督徒需要做计划，但是要把计划放在主的手中，并不是计划了就非要操作，学习时刻顺服神的引导。

或许说这些过于飘渺，但是却要实际操作。因为人计划如何，与人的心思有关系。人类第一个计划就是犯罪计划：『夏娃吃善恶果』。本身这个计划有仇敌的介入，且有夏娃自己的打算，却带全人类进入罪中。所以做计划要做，有几个原则

  • 一，不离开圣经的教训和原则，犯罪之事不用考虑
  • 二，不违背圣经之事，也要随时寻求，要能接受任何时候被神截断计划的可能性，不要对做事的结果有贪恋。
  • 三，在没有领受引导之前当努力执行计划。

圣经里一个著名的例子是使徒保罗的见证。

圣灵既然禁止他们在亚细亚讲道，他们就经过弗吕家、加拉太一带地方。到了每西亚的边界，他们想要往庇推尼去，耶稣的灵却不许。他们就越过每西亚，下到特罗亚去。 (使徒行传 16:6-8 和合本)

这是被神打断传道计划的例子。这些引导在宣教士中特别常见，因为不是凭着血气争战，乃是耶和华的灵成事。也就是即使有时候真的是正确之事，还要顺服圣灵的引导。对于圣灵禁止保罗在亚细亚讲道，保罗选择顺服。

服侍的人与根基

发表于 2018-10-17

服侍的人

然而，难道神只是选召一部分人服侍吗？不是的。所有重生得救之人都被呼召成为服侍的人，所以不存在所谓的平信徒的讲法，该讲法是不符合圣经的。我们可以说神在教会设立了有职分的人来治理神的家，如同圣经所记：传道人，牧师，教师，执事。这些人的确我们要尊重他们，因为他们如同家里位分大的人一样。但并不是说，地位有区别，我们都是神的儿女，这是最为重要的身份。为此，神呼召所有的基督徒都成为服侍的人。

如何去服侍呢？

要心意更新而变化，做什么呢？让我们能够去察验神的旨意。神的什么旨意呢？善良，纯全，可喜悦的旨意。为此，这成为服侍的人的必须，不然很多服侍都不在神的心意之中。既说察验，也就是本身对神的旨意就有一个思考，祷告，执行，判断的过程。而这个过程的根基乃是『不要效法这个世界，只要心意更新而变化』，这两点都需要圣灵的引导和更新，对圣经的阅读和思想，双膝的祷告和寻求。

服侍的根基

为此，服侍之人要蒙主的悦纳，必然是一个和主关系极为亲近之人，那是侍立在耶和华面前最为核心的目的，关系与主越来越美好，因为服侍很容易让我们看工作本身超过主自己，反而就失去了起初的爱，那时候灯台都要被主挪去。所以基督徒服侍。不是搞事业。不是办公司，不是做慈善，不是搞公益，我们一定要定规这些事情的界限，才能不至于非常的迷茫没有方向。

“关系”是教导孩童的核心

发表于 2018-10-17

什么是『世界的智慧』呢？

核心在于有无神的位置在里面。举个例子，如果马哲看似有智慧，辩证法看似有智慧，但因为没有神，其必然导致人自高自大，失去对主的敬畏。

敬畏耶和华是智慧的开端

也就说真智慧不存在在不敬畏神的人身上。圣经从来没有根据一个人的知识量，智商水平，等等现代社会的评判标准去衡量一个人在主面前是否真的有智慧。为此，在训练自己和训练孩子的重心上，要有所取舍。

如何训练

  • 第一，一定是耶稣基督并他钉十字架这个看似『愚拙』的道理，要自己领受，活在生命中，然后教导给孩子。世人会轻看这个『愚拙』的道理，因为心未开启。
  • 第二，在普遍启示的范畴内，教导认识万物和他们的关系，并且要从中显明神在这些被造物上的荣耀，却不能将荣耀归给被造之物。
  • 第三，在『人』的认知上，也要以神为中心。包括：
    1. 人与神的关系最为重要
    2. 人与人的关系需要人与神的关系作为基础，否则为空中楼阁
    3. 人与万物的关系乃是要学习看到神赐给人的治理的权柄和责任
    4. 万物之间的关系乃是看到所有被造物的不可分割又是独立的，此为三位一体的神在被造的万物中的影子。
    5. 所以，当整个被造界的知识体系被建立在以神为中心的基础上，这些都会成为一种预备，可以为主使用。
